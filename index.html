Sobre o Projeto

Este projeto consiste na criação de uma página web estática para a disciplina de Programação para Web. O código-fonte pode ser acessado no repositório do GitHub.

Objetivos do Projeto

O objetivo principal é aplicar os conceitos de HTML semântico e estilização com CSS para criar uma página web bem estruturada e visualmente agradável.

Tecnologias Utilizadas

As seguintes tecnologias foram empregadas no desenvolvimento deste projeto:

HTML5: Para a estruturação semântica do conteúdo da página.
CSS3: Para a estilização, layout e responsividade do site.
